# CLAUDE.md

このファイルは Claude Code がこのプロジェクトを扱う際のガイドです。

## プロジェクトの目的

2026年7月18日(土)〜19日(日)の**草津温泉 1泊2日 旅行**の「旅のしおり」を作り、
友人グループで共有する。成果物は次の2種類。

1. **スマホ最適化のHTMLしおり**（`output/草津温泉_旅のしおり.html`）— 当日スマホで見る本体。
2. **Googleマイマップ用KML**（`output/maps/*.kml`）— 散策ルートをマップに読み込むためのデータ。

最終的にはHTMLを無料の静的ホスティングで**公開**し、スマホのClaudeからも内容を更新できる状態にするのがゴール。

## デザインの方向性

- **トーン**: 温泉旅行らしい和モダンで落ち着いた雰囲気。友人グループ向けの親しみやすさ。
- **配色**（`output/草津温泉_旅のしおり.html` の `:root` CSS変数が正）:
  - 地色 `--bg:#FCF5EF`（クリーム）／文字 `--ink:#1F1B17`（暖かい黒）／補足 `--ink-soft:#6E645B`
  - アクセント（時間厳守）`--accent:#B5471F`（朱）
  - カテゴリ色: 温泉 `--onsen:#9C6B3B`／グルメ `--gourmet:#7A7A3C`／観光 `--sight:#C2703F`／行路・帰路 `--route:#837B70`／予約済み `--reserved:#A3B18A`
- **フォント**: 見出しに Noto Serif JP（明朝）、本文に Noto Sans JP、英字ディスプレイに Mohave。
- **レイアウト**: 幅480pxのスマホ縦想定。1日目・2日目をセクション分けし、各時刻を「時刻｜内容」のタイムライン形式。行路／観光／帰路の小見出しで区切る。アイコンで種別（電車・温泉・食事・観光・バス）を表現。
- 掲載する行程・スポット・料金・参考リンクの一次情報は `docs/design-prompt.txt` を参照。

## 編集ルール / 作業のヒント

- **HTMLしおりが主成果物**。内容変更はまず `output/草津温泉_旅のしおり.html` を直接編集する。要素には編集用のコメント/IDを付与済み。
- ルートを変えたら `src/` のスクリプトを**プロジェクトルートから**実行してKMLを再生成する（例: `python3 src/gen_split.py`）。出力先は `output/` 以下の相対パス。
- ルートは「散策ルート」の呼称で統一（旧称の「グルメ重視ルート」等はファイル名に残るが表示名は散策ルート）。
- 日付は 2026/7/18(土)〜19(日) 固定。
